# commit-push

現在の変更を適切にコミットしてリモートリポジトリにプッシュする。

## 手順

### Step 1: コミット

`~/.config/claude/skills/commit/SKILL.md` を読み、その手順（変更内容の確認 → チケットID要否の判定 → 意味のある単位への分割 → ステージングとコミット）に従ってコミットする。コミットメッセージ形式・チケットID対応のルールもすべてそちらに従うこと。

### Step 2: リモートリポジトリへプッシュ

コミットが完了したら、まとめてプッシュする。

```bash
git push
```
